### 1. **SISTEMA DE TAGS/CATEGORIAS**
```sql
-- Nova tabela
CREATE TABLE tags (
    id INT AUTO_INCREMENT PRIMARY KEY,
    nome VARCHAR(100) NOT NULL UNIQUE,
    cor VARCHAR(7) DEFAULT '#007bff'
);

CREATE TABLE album_tags (
    album_id INT,
    tag_id INT,
    PRIMARY KEY (album_id, tag_id)
);
```
**Benefício:** Organizar álbuns por "Casamento", "Aniversário", "Viagem", etc.

## 💻 CÓDIGO DE EXEMPLO - SISTEMA DE TAGS

```php
// Em functions.php
function getAlbumTags($album_id) {
    $db = getDB();
    return $db->fetchAll("
        SELECT t.* FROM tags t 
        JOIN album_tags at ON t.id = at.tag_id 
        WHERE at.album_id = ?
    ", [$album_id]);
}

function addTagToAlbum($album_id, $tag_name, $color = '#007bff') {
    $db = getDB();
    
    // Criar tag se não existir
    $tag_id = $db->insert("
        INSERT IGNORE INTO tags (nome, cor) VALUES (?, ?)
    ", [$tag_name, $color]);
    
    if (!$tag_id) {
        $tag = $db->fetchOne("SELECT id FROM tags WHERE nome = ?", [$tag_name]);
        $tag_id = $tag['id'];
    }
    
    // Associar ao álbum
    return $db->insert("
        INSERT IGNORE INTO album_tags (album_id, tag_id) VALUES (?, ?)
    ", [$album_id, $tag_id]);
}
```
